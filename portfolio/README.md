# Portfolio (Arindam Kumar Nath)

Run locally:
- Open `portfolio/index.html` in a browser, or
- Serve the folder: `python3 -m http.server -d portfolio 5173` then visit `http://localhost:8000`.

Edit:
- Update `portfolio/index.html` JSON block (id: `resume-data`) to change content.
- Tweak styles in `portfolio/styles.css`.
- Card topics live in `portfolio/app.js` under `topics`.
